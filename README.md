SMS Spam Detection System
Overview

The SMS Spam Detection System is a Machine Learning-based project that classifies SMS messages as Spam or Not Spam (Ham) using Natural Language Processing (NLP) techniques.

This project helps in automatically filtering unwanted or fraudulent messages, improving user experience and security.
